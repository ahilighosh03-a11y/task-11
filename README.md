# Task 12: Compound Interest Calculator

A simple web-based program that automatically calculates compound interest based on predefined static values and displays the final result in the browser console.

## Project Structure
The project consists of the following files:
* `index.html` - The webpage structure.
* `style.css` - Basic formatting and centered layout styling.
* `script.js` - The JavaScript logic containing the static variables and compound interest formula.
* `Readme.md` - Documentation for the assignment submission.

## Objective & Formula
The program uses the standard compound interest formula to compute the final balance and total interest earned:

**Formula:** A = P(1 + (r / n))^(n * t)

* **P (Principal):** 42,200
* **r (Annual Interest Rate):** 0.058 (5.8%)
* **n (Compounding Periods per Year):** 4 (Quarterly)
* **t (Time in Years):** 3

## How to Run the Project
1. Open the project folder on your computer.
2. Double-click the `index.html` file to launch it in any modern web browser (such as Chrome, Edge, or Safari).
3. Right-click anywhere on the webpage and select **Inspect**.
4. Go to the **Console** tab to view the printed output.

## Expected Output
The console will log the calculated value in the following format:
`The compound interest after 3 years is: 7958`
